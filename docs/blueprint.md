# **App Name**: HDS DomainAI Discover

## Core Features:

- User Input Form: Collect user data including business/personal type, project name, niche, target audience, keywords, and preferred TLDs.
- AI-Powered Domain Suggestion: Utilize the Manus AI API to analyze user inputs and suggest the top 3-5 most suitable domain names based on market trends, branding potential, and audience relevance. Include confidence scores and explanations for each.
- Domain Availability Check: Enable users to check domain availability via a button that triggers the check in WHMCS
- WHMCS Cart Prefill: Integrate a feature to prefill the WHMCS cart with the selected domain for easy registration.
- CRM Integration: Capture user information and selected domain to update the CRM and tag workflows for business or personal domains, triggering automated follow-ups.
- Result Display: Display suggested domains with confidence scores, explanations, and a button to prefill the WHMCS cart, plus JS snippet to prefill and trigger the availability check.
- Funnel Integration placeholders: Comments added as a tool, as hints for the student who needs to connect the app to CTA buttons.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), suggesting reliability, innovation, and intelligence.
- Background color: Light gray (#F5F5F5) for a clean, modern interface, providing a neutral backdrop that enhances readability and focus.
- Accent color: Warm orange (#FFB347), used strategically for call-to-action buttons, highlighting the path to domain registration. This helps attract user attention to the most important actions within the application, promoting conversions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines to give a modern, techy feel, paired with 'Inter' (sans-serif) for body text to ensure readability.
- Use minimalist icons related to domains, search, and business, maintaining a clean and modern aesthetic.
- A clean and structured layout to guide users through the domain discovery process, making information accessible.
- Subtle animations during the domain generation and loading process to improve user engagement and provide feedback.
